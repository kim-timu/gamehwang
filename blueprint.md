# Project Blueprint

## Overview

This project is a web-based game arcade. It starts with a lobby where users can select a game to play. The initial game offered is a classic retro-style Tetris.

## Project Structure

- `index.html`: The main entry point of the application, serving as the game selection lobby.
- `tetris.html`: The page dedicated to the Tetris game.
- `style.css`: Contains the styles for the application, including the game lobby and the Tetris game.
- `main.js`: The core logic for the Tetris game.

## Current Features

- **Game Lobby:** A simple and intuitive interface to select a game.
- **Tetris Game:** A fully functional, retro-style Tetris game with scoring, levels, and a next-block preview.
- **Responsive Design:** The layout adapts to different screen sizes for a consistent experience on desktop and mobile.

## Design and Style

- **Theme:** Retro arcade theme with pixel fonts and a dark color scheme.
- **Visuals:** Use of box shadows and a clean layout to create a modern, lifted look.
- **Interactivity:** Game cards in the lobby have hover effects to provide visual feedback to the user.
